1. What is a neural network?¶

A neural network takes an input, passes it through multiple layers of hidden neurons (mini-functions with unique coefficients that must be learned), and outputs a prediction representing the combined input of all the neurons. Neural Networks consist of the following components

1-) An input layer, x
2-) An arbitrary amount of hidden layers
3-) An output layer, ŷ
4-) A set of weights and biases between each layer, W and b
5-) A choice of activation function for each hidden layer.

1.1 Example of a neural network architecture: 4-Layer neural network¶

Let us construct a four-layer network (input, 2-hidden layers, output) Source: [https://towardsdatascience.com/]

1-) Input layer - 64 neurons (input image array)
2-)Hidden layer 1 - 128 neurons (arbitrary)
3-) Hidden layer 2 - 128 neurons (arbitrary)
4-) A set of weights and biases between each layer, W and b
5-) Output layer - 10 neurons (output one-hot array)

3. Feed forward part¶

3.1 Initialize the weights and biases¶

1-) Randomly initialize the weights. The values should never be zero but closer to zero. 
2-) It is good to have positive as well as negative values so that apart from magnitude, direction also differs. 
3-) Normal distribution is a good choice. 
4-) Initializing bias vectors to zero is fine.

3.2 Feed-forward process¶

$${\displaystyle c={{(input \cdot weights) + bias}}} \hspace{1cm} {\displaystyle a1= f1(c)}$$

def feed_forward(x, Wh, Wo):
    # Hidden layer
    Zh = x * Wh
    H = activation_function(Zh)

    # Output layer
    Zo = H * Wo
    output = activation_function(Zo)
    return output

4. Backpropogation¶

Back-propagation computes the error from the output of feed-forward and the true value. This error is back-propagated to all the weight matrices through computing gradients in each layer and these weights are updated.

4.1 Loss function¶

Generally speaking, the loss function is designed to show how far we are from the 'ideal' solution.

class neural_networks:

        def __init__(self, x, y, layers=[128, 128,10], **kwargs):

            #self.activation_name = kwargs.pop('activation_name', 'ReLu')
            self.cost_name = kwargs.pop('cost_function_name', 'cross_entropy')
            self.input= x
            self.output=y 
            #learning rate 
            self.learning_rate=0.5
            super(neural_networks, self).__init__()
            #the size of the input
            self.input_size=x.shape[1]
            #the size of the output 
            self.output_size=y.shape[1]
             #initialize the weights for one layer, matrix (input_layer, output_layer) 
            self.weights = [np.random.randn(self.input_size, layers[0])]
            #initialize the biases for one layer, vector (1, output_layer)
            self.biases=[np.zeros((1,layers[0]))]

            for i in range (len(layers)-1):
                self.weights.append(np.random.randn(layers[i], layers[i+1]))
                self.biases.append(np.zeros((1,layers[i+1])))
            self.weights=np.asarray(self.weights)
            for i in range (len(layers)):
                self.biases[i]=np.asarray(self.biases[i])

        def compute_activ_func(self, x, activation_name): 
            self.activation_name=activation_name
            if self.activation_name=="ReLu": 
                # Rectified linear function, which outputs the max between 0 and x 
                y=np.maximum(0,x)
                y_prime=np.where(x <= 0, 0, 1)
            elif self.activation_name=="Sigmoid": 
                    #Sigmoid function, squishes the input between 0 and 1 
                y=1/(1 + np.exp(-x))
                y_prime=y * (1 - y)
            elif self.activation_name=="Softmax": 
                  #normalizes the input into a probability distribution consisting of K probabilities, which sum to 1
                exps = np.exp(x - np.max(x, axis=1, keepdims=True))
                y=exps/np.sum(exps, axis=1, keepdims=True)
                y1=y.reshape(-1,1)
                y_prime= np.diagflat(y1) - np.dot(y1, y1.T)
            elif self.activation_name=="linear": 
                y=x 
                y_prime=1
            elif self.activation_name=="tanh":
                # tanh function outputs number between -1 and 1 
                y=np.tanh(x)
                y_prime=1-np.tanh(x)**2
            else: 
                raise ValueError('Activation function is not supported')
            return y, y_prime 

        
        def compute_loss_func(self, y_pred, y_real): 
            if self.cost_name=="MSE":
                e=np.mean(np.power(y_real-y_pred, 2))
                e_prime= 2*(y_pred-y_real)/y_real.size
            #−(ylog(p)+(1−y)log(1−p))
            # −∑yo,clog(po,c)
            elif self.cost_name=="cross_entropy":     
                n_samples = y_real.shape[0]
                #y_real.argmax(axis=1) returns the indices of 
                logp = - np.log(y_pred[np.arange(n_samples), y_real.argmax(axis=1)])
                #log_likelihood = -np.log(p[range(a),y_true])
                e = np.sum(logp)/n_samples
                #p[range(a),y_true] -= 1
                e_prime = 0
            else:
                raise ValueError('Cost function is not supported')
            return e, e_prime
  
        

        def feedforward(self):
            z1 = np.dot(self.input, self.weights[0]) + self.biases[0]
            self.a1= self.compute_activ_func(z1, activation_name="Sigmoid")[0]
            z2 = np.dot(self.a1, self.weights[1]) + self.biases[1]
            self.a2=self.compute_activ_func(z2, activation_name="Sigmoid")[0]
            z3 = np.dot(self.a2, self.weights[2]) +self.biases[2]
            self.a3=self.compute_activ_func(z3, activation_name="Softmax")[0]
        def backpropagation(self):
            loss=self.compute_loss_func(self.a3, self.output)[0]
            print('Error :', loss)
            a3_delta = (self.a3- self.output)/self.output.shape[0] # w3
            z2_delta = np.dot(a3_delta, self.weights[2].T)
            a2_delta = z2_delta * self.compute_activ_func(self.a2, activation_name="Sigmoid")[1] # w2
            z1_delta = np.dot(a2_delta, self.weights[1].T)
            a1_delta = z1_delta * self.compute_activ_func(self.a1, activation_name="Sigmoid")[1] # w1

            self.weights[2] -= self.learning_rate * np.dot(self.a2.T, a3_delta)
            self.biases[2] -= self.learning_rate * np.sum(a3_delta, axis=0, keepdims=True)
            self.weights[1] -= self.learning_rate * np.dot(self.a1.T, a2_delta)
            self.biases[1]  -= self.learning_rate * np.sum(a2_delta, axis=0)
            self.weights[0] -= self.learning_rate * np.dot(self.input.T, a1_delta)
            self.biases[0] -= self.learning_rate * np.sum(a1_delta, axis=0)
        def train(self, epochs, **kwargs):
            for i in range(epochs):
                self.feedforward()
                self.backpropagation()
            
        def prediction(self, new_data):
            self.input=new_data 
            self.feedforward()
            predictions= self.a3.argmax()
            return predictions

        def get_acc(self, x, y):
                acc = 0
                for xx,yy in zip(x, y):
                    s = self.prediction(xx)
                    if s == np.argmax(yy):
                        acc +=1
                return acc/len(x)*100
